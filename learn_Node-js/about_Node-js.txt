
Node.jsとは？
Node.jsとは従来Webブラウザなどクライアント側で動作していたJavaScriptを、
サーバー側でも動作できるようにするためのプラットフォームです。

Node.jsフレームワーク一覧
・Express
Express.jsはバックエンドの部門で5年連続の利用率1位となった、
最古参で人気があるNode.jsフレームワーク
・Vue.js
Vue.js」は数あるなかでもUI構築用として高い人気を誇るNode.jsフレームワークの1つです。
・Angular
Angularは、フロントエンド開発で必要な機能がそろったフルスタックフレームワークです。
AngularはGoogle社が主導となって開発が進められており、オープンソースとして公開されています。
Angularの大きな特徴としてあげられるのは、あらゆるプラットフォームに対応している点です。
AngularであればPC用・スマートフォン用のウェブ、PC・スマートフォンのネイティブアプリなど
あらゆる環境向けの開発が行えます。
・React
「React」はMeta（旧Facebook）社が開発したUI構築用のNode.jsライブラリ※です。
高速に動作する点が大きなメリットと言えます。Reactは、昨今流行しているSPA
（Single Page Application）の開発にも最適です。
・Next.js
Next.js = Reactを拡張するのに使われるフレームワーク
UI構築用であるReactにNext.jsを組み合わせることで、
サーバーサイドでも動作するReactアプリを開発できるようになります。

・npmとは？
(Node Package Manager)
Node.jsには、モジュール化といってさまざまな機能を持ったプログラムを
個別のファイルに分割することができます。
このような世界中で公開されてる便利なモジュールを管理できるツールが、npmです。
